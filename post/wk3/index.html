Lists

Lists are useful when we need to perform analysis over heterogeneous pieces of information. Lists are also useful to store different classes of information together. Most R functions output their results in lists.

A list has a length, but no dimension!

  • Entries of a list can be numbers, strings, logics, vectors, matrices, data frames, or lists as well.

empty_list <- list()

#verify that empty_list is of class 'list'
class(empty_list)

#view length of list
length(empty_list)

[1] 0

empty_list[[1]] <- "Aaron"
empty_list[[2]] <- "56"

empty_list[[4]] <- matrix(c(0.23, 0.56, 0.30, 0.22, 0.36, 1, 2, 2, 1, 1), nrow=5)

#or equivalently, you can use the following to assign values to a list
empty_list[1] <- "Aaron"
empty_list[2] <- "56"
empty_list[3] <- TRUE
empty_list[4] <- list(matrix(c(0.23, 0.56, 0.30, 0.22, 0.36, 1, 2, 2, 1, 1), nrow=5))

tumor_sizes_factor2 <- factor(tumor_sizes, ordered=TRUE, levels=c("Small", "Medium", "Big"))
[1] Small  Big    Big    Medium Medium Small  Medium Small  Small 
Levels: Small < Medium < Big

### Notice that as.factor() function does not have the options "ordered=, levels="!!
